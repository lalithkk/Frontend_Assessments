# 🌟 1.HastRekha – AI Palm Reading Landing Page

PalmRekha is a modern, cinematic AI-powered palm reading landing page built using React, Vite, Tailwind CSS, and Framer Motion.
The project focuses on immersive storytelling, elegant animations, interactive palm exploration, and premium UI/UX design.

---

## 🚀 Live Features

* Responsive premium landing page
* Animated SVG palm illustration
* Interactive palm line explorer
* Scroll reveal animations
* Animated counters
* Custom gold cursor effect
* FAQ accordion section
* Testimonials carousel
* Fully responsive mobile-first design
